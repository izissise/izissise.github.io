Kde Kwin Overview Gesture Patch
Posted on 3024-03-08 DRAFT

Plasma 6

I recently updated to Kde Plasma 6, it is great except for one nitpick, the overview effect is trigger on the touchpad with 4 fingers and I'd like to trigger it with only 3.

And the value is (hardcoded in the C++ code)[https://invent.kde.org/plasma/kwin/-/blob/master/src/plugins/overview/overvieweffect.cpp#L36]

It seems easy enough to patch the binary, let's try

Analyze

Using objdump -D /usr/bin/kwin_wayland | grep -i touchpad we can find that the method addTouchpadSwipeGesture is indeed in the kwin_wayland binary, with the mangled name _ZN4KWin22EffectTogglableGesture23addTouchpadSwipeGestureENS_14SwipeDirectionEj and we find exactly 3 calls to it corresponding to the source code.

Using objdump -D we can extract the hex form of the instructions for fingerprinting, keeping only arguments to the call instruction.

Example

With:

   d7b58:       48 89 ef                mov    %rbp,%rdi
   d7b5b:       ba 04 00 00 00          mov    $0x4,%edx
   d7b60:       be 03 00 00 00          mov    $0x3,%esi
   d7b65:       e8 06 dd f7 ff          call   55870 <_ZN4KWin22EffectTogglableGesture23addTouchpadSwipeGestureENS_14SwipeDirectionEj@plt>

we want 4889efba04000000be03000000e8

We end up with 2 fingerprints for the 3 calls

4889efba04000000be03000000e8
4889efba04000000be01000000e8

We can check that it doesn't match anything else

$ xxd -p -c 0 /usr/bin/kwin_wayland | tr -d '[:space:]' | grep -Eo '4889efba04000000be0[3,1]000000e8'
4889efba04000000be03000000e8
4889efba04000000be03000000e8
4889efba04000000be01000000e8

Patching

We got our fingerprints we can patch the binary, to have 3 finger gesture instead of 4 we can just replace mov $0x4,%edx with mov $0x3,%edx.

The patching is the easier step, first backup the original binary

sudo cp -a /usr/bin/kwin_wayland /usr/bin/kwin_wayland.bak

and then patch using xxd and sed:

xxd -p -c 0 /usr/bin/kwin_wayland | tr -d '[:space:]' | sed 's|4889efba04000000be0\(.\)000000e8|4889efba03000000be0\1000000e8|g' | xxd -r -p > kwin_wayland_patched
# replace binary
sudo mv kwin_wayland_patched /usr/bin/kwin_wayland && sudo chmod 755 /usr/bin/kwin_wayland && sudo chown root:root /usr/bin/kwin_wayland

Reboot and enjoy!
