Linux basic initramfs

Let us create an initramfs using the official documentation

initramfs_list

initramfs_list is a simple tool that takes an initramfs description and create the archive.

Tool compilation

It needs some preparations, we need to files that can found in the kernel repository

  • gen_initramfs.sh
  • gen_init_cpio.c

gen_init_cpio.c needs to be compiled

gcc -o gen_init_cpio gen_init_cpio.c

gen_initramfs_list.sh make the assumption that gen_init_cpio is in a directory 'usr', lets assume the current directory instead

sed -i -E "s|usr/gen_init_cpio|${PWD}/gen_init_cpio|g" gen_initramfs.sh

Now we are ready to use it.

Basic Initramfs

We can describe a basic initramfs using the following

cat > cpio_list <<EOF
dir . 1755 0 0
dir proc 1775 0 0
dir dev 1775 0 0
dir bin 1775 0 0
dir sys 1775 0 0
dir run 1775 0 0
dir mnt 1775 0 0
dir mntroot 1775 0 0
dir dev/pts 1775 0 0
dir dev/shm 1775 0 0
dir dev/udev 1775 0 0
nod dev/zero 1666 0 0 c 1 5
nod dev/null 1666 0 0 c 1 3
nod dev/console 1622 0 0 c 5 1
nod dev/loop0 1644 0 0 b 7 0
nod dev/ttyS0 1666 0 0 c 4 64
nod dev/tty 1666 0 0 c 5 0
nod dev/ptmx 1666 0 0 c 5 2
nod dev/random 1444 0 0 c 1 8
nod dev/urandom 1444 0 0 c 1 9
slink dev/core /proc/kcore 1777 0 0
slink dev/fd /proc/self/fd 1777 0 0
slink dev/stdout /proc/self/fd/1 1777 0 0
slink dev/stdin /proc/self/fd/0 1777 0 0
slink dev/stderr /proc/self/fd/2 1777 0 0
slink bin /sbin 1777 0 0
slink init /bin/init 1777 0 0
file bin/init initloader 0755 0 0
EOF

this is line separated description of the initramfs, each line start with a file type, path, permissions, major and minor. type file also takes an existing file as parameter, here we use it to set init binary (start program). type slink also take the symbolic link target path.

We now have cpio_list file describing our initramfs.

But we can't build it just yet, we also need create an init program.
